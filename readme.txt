== Description ==

"Negotiabl for WooCommerce" helps WooCommerce store owners drive more sales. On cart inactivity, the customer on your store is shown a popup asking him to start a live negotiation with the store owner on web chat or on Facebook Messenger. Once agreement is reached a coupon is automatically generated and the customer can make his purchase instantly.

== Installation ==

1. Install the plugin from the plugin directory
2. Register Store:
2.a. Signup your store on negotiabl.com <a href="http://negotiabl.com">Setup your store & get Credentials</a>
2.b. Make a note of AppID & Password
3. Plugin Activation:
3.a. Navigate to WooCommerce>Settings>Negotiabl
3.b. Enter AppID & Password, Save Changes
4. That is it. Your store is now connected to Negotiable
5. Test it out by adding an item to your store cart
6. Log into the <a href="https://app.negotiabl.com/">Negotiable dashboard</a> to respond to live negotiations.

= Can I control when the popup shows up? =

Yes, you can set the cart inactivity time after which a popup is shown from the app dashboard.

= Can I chat with the customer? =

Absolutely! Chat from the app dashboard. Sell yourself and your store.

= Can I message the customer?  =

As per Facebook rules, you have 24 hours from their last message/activity to reach out to them and try to close the deal.
